You can use the * operator to repeat a certain collection multiple times, creating a new collection that contains the original repeated. This operation is valid for ordered collections like strings, lists and tuples. Here, we'll use lists and strings as an example.

Basic Usage

print([0, 1] * 3) equals_string="=" print(equals_string * 5)

Practical Uses

String repetition is useful for creating patterns, separators, or formatting:

# Create a divider line print("-" * 40) # Create padding spaces = " " * 10 print("|" + spaces + "centered" + spaces + "|")

Order Doesn't Matter

You can place the number before or after the collection:

print("!" * 5) print(5 * "!")
