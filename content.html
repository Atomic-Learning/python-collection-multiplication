You can use the * operator to repeat a string multiple times, creating a new string that contains the original repeated.

Basic String Repetition

print("ha" * 3) print("=" * 5)

Using Variables

You can repeat strings stored in variables:

word = "echo " repeated = word * 4 print(repeated) print(word) # The original string is unchanged

Practical Uses

String repetition is useful for creating patterns, separators, or formatting:

# Create a divider line print("-" * 40) # Create padding spaces = " " * 10 print(spaces + "centered")

Order Doesn't Matter

You can place the number before or after the string:

print("!" * 5) print(5 * "!")
